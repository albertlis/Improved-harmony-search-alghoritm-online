The user defined parameters class initializer.

Parameters
[in]parametersThe dictonary with user defined parameters.
Returns
An instance of the IHSParameters class initialized with the specified parameters.
Warning
The parameters dictonary has to have 9 key-vaule pairs. Example:
{
'function': 'x1 + x2',
'iterations': 50000,
'hms': 10,
'hcmrmin': 0.5,
'hcmrmax': 0.75,
'parmin': 0.2,
'parmax': 0.8,
'bwmin': 0.1,
'bwmax': 2.0
}
